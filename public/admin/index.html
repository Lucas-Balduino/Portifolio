Admin — Gerenciar Projetos (modo estático)

Interface administrativa para criar, editar e remover projetos usando um arquivo JSON estático versionado no GitHub. Não há servidor ou banco de dados. Faça as alterações, gere o JSON atualizado e faça commit/push para publicar.

Usado na URL: /projeto.html?slug=meu-projeto-incrivel
