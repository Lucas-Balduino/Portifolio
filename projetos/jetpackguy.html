JetPackGuy

Jogo 2D - controles por teclado e física simples

Introdução

Este projeto nasceu como um exercício para aprender animações frame-by-frame e manipulação de eventos de teclado no browser. O objetivo principal foi construir um protótipo jogável com baixa latência e controles responsivos.

Tecnologias

  • HTML5 Canvas
  • JavaScript puro
  • CSS para UI

Galeria

Desafios e soluções

Problemas com detecção de colisões e jitter foram resolvidos simplificando a física e limitando a taxa de atualização. Para aprofundar, experimentei técnicas de interpolação para suavizar o movimento.

Como rodar

Abra a pasta do projeto e abra o arquivo index.html no navegador. Para executar localmente com servidor: npx http-server .
